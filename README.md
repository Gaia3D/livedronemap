# LiveDroneMap
 UAV 기반 매핑 및 공유 솔루션
 
### Feature

1.드론 영상 자료 수집 기능
- 드론 영상자료의 모니터링
- 멀티 드론 영상 자료 전송

2.드론 영상 처리 기능
- 좌표 변환 및 다축척 영상 생성

3.드론 영상 가시화 기능
- 웹 기반 맵 도시
- 스마트 줌

4.선박 및 유류유출 탐지 결과 가시화 기능

5.후처리 결과 가시화 기능

### Development Environment

- Spring
- mybatis
- lombok
- PostgreSQL
- PostGIS
- GeoServer
- Gradle
