
 - 선박 탐지 결과는 다음과 같은 속성으로 이루어저 있다.
  1. 선박에 해당하는 부분을 bounding box로 표현한 가시광 혹은 열화상 이미지
  2. 선박의 위치
  3. bounding box의 면적
  4. bounding box의 장축의 길이
  5. bounding box의 단축의 길이
  
  bounding box = Geometry(폴리곤)형태로 json으로 받는다.
  
  1. 선박의 좌표값에 해당 되는 위치에 폴리곤을 그린다.
  	1.1 이전 이미지와 현재 이미지의 같은 선박 ID의 선박일 경우에는, 현재의 이미지에만 폴리곤을 그린다.
  	1.2 선박의 이전 경로를 살펴볼 경우에는 탐지결과 상세에서 이전 경로를 클릭한다. 
  2. 해당 위치의 폴리곤 클릭시 속성정보 보기
    2.1 속성 정보에는 다음과 같은 정보를 나타낸다.
     - 선박 ID
     - 선박 위치
     - 면적(Object Area)
     - 장축의 길이(Major Axis)
     - 단축의 길이(Minor Axis)
    2.2 버튼(이전 경로)
